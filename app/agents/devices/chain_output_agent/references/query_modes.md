# Query Mode Rules

## Discovery Mode
User is exploring options. Use a table to help them compare. Present all options neutrally without ranking or preference.

## Comparison Mode
User is comparing options. Use a side-by-side table showing specifications. Present differences objectively without recommending one over another.

| Spec | Option A | Option B |
|------|----------|----------|
| ID | 0.021" | 0.017" |
| OD | 0.026" | 0.029" |
| Length | 150cm | 150cm |

Let the specifications speak for themselves - do not state which is "better".

## Default Mode
No special mode adjustments needed.
